SKYNET

SYNOPSIS

Synrc Skynet is a specialized distribution application for drone swarm coordination implementing Link32 tactical battlefield protocol inspired by VMF, TSM, SRW formats with MQTT and LTE 4G (HSS, PDCP, QCI) protocol stacks designed for military applications requiring low-latency, secure, and scalable data exchange in contested environments. It supports swarm coordination, real-time position location information (PLI), command and control (C2), and tactical chat over UDP-based multicast networks.

FEATURES

  • ECDH
  • SECP384R1
  • CMPXCHG CAS
  • UDP
  • RTOS
  • MULTICAST
  • LTE (srsRAN-4G)
  • FN1V

SKYNET

Skynet command line interface (CLI) is set of POSIX tools running Synrc real-time system (RTS), a NuttX derivative work like Sony Spresense.

KEYGEN

Generates ECC secp384r1 key pairs for the specified node, storing them in ~/.skynet/ecc/secp384r1/ (server) or ~/.skynet_client/ecc/secp384r1/ (client).

ENCRYPT

Encrypts a test message from <sender> to <recipient> for the specified NPG, producing <npg_id>.sky.

DECRYPT

Decrypts <file> (e.g., <npg_id>.sky) using keys for <sender> and <recipient>.

SERVER

Runs the server named as , FNV1A 32-bit hash is used. The server assigns a slot, forwards status messages to 239.255.1., and logs all activity.

CLIENT

Runs the client named as , FNV1A 32-bit hash is used. The client sends key exchange, slot request, and status messages.

SIM FLASH (PCSC/CCUD) and eSIM QR generator

Utility to flash SIM cards for srsRAN setups with PC/SC and CCID support. It uses libpcsclite, libusb-1.0 as dependencies and supports PCSC (OMNIKEY, ACR38U), CCID devices (Автор).

eSIM SM-DP+ server for iOS

Lightweight SM-DP+ server for eSIM profile delivery in srsRAN 4G. Generates GSMA-compliant eSIM profiles for iPhone SE 5G.
